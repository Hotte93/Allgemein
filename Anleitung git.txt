Git Commands

	Dokumente herunterladen
	
		git pull <url> master  //Lädt alle Änderungen im Repository herunter
	
	Dokumente hochladen
	
		git add <Dokument> //Änderungen im lokalen Repository speichern
		git commit-m <Dokument> //Änderungen im Dokument werden (-m zeigt die Message dazu)
		git push <url> master //Dateien werden auf Remote Repository geschrieben
